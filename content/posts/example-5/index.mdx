## Using shortcuts
You can enter and edit quite complex equations using just a few shortcuts 


![](attachments/short1.gif)<wljs-html encoded="true">{`%3Cbr%20%2F%3E`}</wljs-html>


- `Alt+2`, `Cmd+2` hide/show input cell
- `Ctrl+/` make fraction on selected
- `Ctrl+6` make superscript on selected
- `Ctrl+2` make square root on selected
- `Ctrl+-` make subscript on selected

- `Ctrl+=` make input form for semantic interpretation


This is a syntax sugar borrowed from Mathematica, but implemented in a much simpler way

<wljs-editor display="codemirror" type="Input"  >{String.raw`(*FB[*)((1)(*,*)/(*,*)(2))(*]FB*)`}</wljs-editor>

If you copy and paste into a normal text editor, what you see is a normal Woflram Language code

<wljs-editor display="codemirror" type="Input"  >{String.raw`(*FractionBox[*)((1)(*,*)/(*,*)(2))(*]FractionBox*)`}</wljs-editor>

Wolfram Kernel can generate them as well like you can expect in Mathematica

<wljs-editor display="codemirror" type="Input"  >{String.raw`Series[Sinc[(*SqB[*)Sqrt[x](*]SqB*)], {x,0,4}] // Normal `}</wljs-editor>

<wljs-editor display="codemirror" type="Output"  >{String.raw`1-(*FB[*)((x)(*,*)/(*,*)(6))(*]FB*)+((*FB[*)((1)(*,*)/(*,*)(120))(*]FB*)) ((*SpB[*)Power[x(*|*),(*|*)2](*]SpB*))-(*FB[*)(((*SpB[*)Power[x(*|*),(*|*)3](*]SpB*))(*,*)/(*,*)(5040))(*]FB*)+(*FB[*)(((*SpB[*)Power[x(*|*),(*|*)4](*]SpB*))(*,*)/(*,*)(362880))(*]FB*)`}</wljs-editor>

<wljs-editor display="codemirror" type="Input"  >{String.raw`n = (*SqB[*)Sqrt[1 - (*FB[*)(((*SpB[*)Power[(*SbB[*)Subscript[\[Omega](*|*),(*|*)p](*]SbB*)(*|*),(*|*)2](*]SpB*))(*,*)/(*,*)(((*SpB[*)Power[(*SbB[*)Subscript[\[Omega](*|*),(*|*)0](*]SbB*)(*|*),(*|*)2](*]SpB*) - 4Pi (*SpB[*)Power[\[Eta](*|*),(*|*)2](*]SpB*) ) - 2 Pi I \[Gamma] \[Eta]))(*]FB*)  ](*]SqB*)  ;`}</wljs-editor>

## Using command palette
One can use a helper palette - `Special characters` panel for typing various mathematical operations


![](attachments/Screenshot%202024-09-03%20at%2011.29.54.png)


and then


![](attachments/short2.gif)

For example

<wljs-editor display="codemirror" type="Input"  >{String.raw`(*TB[*)Integrate[(*|*)Sin[x](*|*), (*|*)x(*|*)](*|*)(*1:eJxTTMoPSmNiYGAo5gESnnklqelFiSWpTvkVmYwgUVYg4ZaYU5wKAMfBCcU=*)(*]TB*)`}</wljs-editor>

<wljs-editor display="codemirror" type="Output"  >{String.raw`-Cos[x]`}</wljs-editor>

Or with defined boundaries

<wljs-editor display="codemirror" type="Input"  >{String.raw`(*TB[*)Integrate[(*|*)Sin[x](*|*), {(*|*)x(*|*),(*|*)B(*|*),(*|*)A(*|*)}](*|*)(*1:eJxTTMoPSmNiYGAo5gESnnklqelFiSWpTvkVmYwgURYgEVJUmgoAvmMJeQ==*)(*]TB*)`}</wljs-editor>

<wljs-editor display="codemirror" type="Output"  >{String.raw`-Cos[A]+Cos[B]`}</wljs-editor>

One can also use `InputForm` for taking integrals

<wljs-editor display="codemirror" type="Input"  >{String.raw`Integrate[Sin[x], x]`}</wljs-editor>

<wljs-editor display="codemirror" type="Output"  >{String.raw`-Cos[x]`}</wljs-editor>
